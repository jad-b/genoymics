Created for Oy* team of MCBS 913, Spring '13 at UNH
Created by Jeremy Dobbins-Bucklad


Purpose:
--------
Provides statistics on taxonomic assignment files generated from OUT maps.


Instructions:
-------------
The script can be run as:

	perl classifs_percent.pl taxonomic_assignments.txt > output.txt

The "> output.txt" is optional, and will redirect the output from the terminal to a file of your choice.

To test using the sample assignments from the QIIME tutorial, run:
	perl classifs_percent.pl tutorial_tax_assns.txt


Requirements: 
-------------
*Perl (Tested on v5.14.2)
